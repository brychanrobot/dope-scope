# DopeScope

A web GUI for USB oscilloscopes such as Owon VDS1022 or Hantek 6022BE. This uses the Web USB API, which should allow users to view the data from the oscilloscopes on any device that can run the Chrome browser. As other browsers begin to support the API, it should work on them as well.
